# AstroSage Call Center Operational Analysis 📊

## Project Overview
This project involves a comprehensive data analysis of **AstroSage**, a leading Vedic astrology platform. The goal was to analyze 28,000+ call records to identify operational bottlenecks and provide a strategic roadmap for a **₹1 Crore investment** to improve efficiency and customer satisfaction.

## 🛠 Tools Used
* **Microsoft Excel / Google Sheets:** Data Cleaning, Pivot Tables, and Dashboards.
* **Statistical Analysis:** Correlation (CORREL), Trend Analysis.
* **PowerPoint:** Strategic Presentation.

## 📈 Key Insights & Findings
* **User Retention:** Discovered an **81.15% repeat caller rate**, indicating high brand loyalty but a heavy reliance on existing users.
* **Peak Performance:** Identified peak call volumes between **5 AM and 11 AM**, suggesting a need for better shift management.
* **Service Quality:** Found a **correlation of 0.05** between call duration and ratings, proving that longer calls do not necessarily mean happier customers.
* **Revenue Drivers:** "Call" consultations significantly outperformed "Chat" in net revenue.

## 🧹 Data Cleaning Process
The raw dataset was expanded from **35 to 43 attributes** to allow for deeper analysis:
1.  **Imputation:** Handled missing `chatStatus` and `chatSeconds`.
2.  **Transformation:** Converted timestamps into `Month`, `Day`, and `Time Slots`.
3.  **Categorization:** Created "Caller Type" (New vs. Repeat) using `IF` and `COUNTIF` functions.

## 💰 Strategic Investment Plan (₹1 Crore)
Based on the data, the following allocation was recommended:
* **60% (₹60L):** Call Center Technology Upgrades (to reduce "failed/incomplete" calls).
* **30% (₹30L):** Guru Training Programs (focusing on skill consistency).
* **10% (₹10L):** Workforce Management Tools (to prevent agent burnout).

## 📂 Project Structure
* `Data/`: Contains the cleaned dataset.
* `Analysis/`: The Excel workbook with Pivot Tables and Charts.
* `Presentation/`: The final PPTX summarizing the strategy.
